Supported Commands:

  1. Canvas & Pointer commands:
    • home -> center turtle
    • cs -> clear screen
    • pu -> pen up
    • pd -> pen down
    • ht -> hide turtle
    • st -> show turtle
    • fill -> fill region
  2. Unary commands:
    • fd -> forward (eg: fd 10)
    • bk -> backward (eg: bk 10)
    • rt -> turn right (eg: rt 90)
    • lt -> turn left (eg: lt 90)
    • seth -> set heading (eg: seth 45)
    • setpencolor -> sets the pen color (eg: setpencolor "blue)
    • setfillcolor -> sets the fill color (eg: setfillcolor "blue)
  3. Binary commands:
    • repeat -> repeats a set of command several times (eg: repeat 4[fd 10 rt 90])
    • setxy -> sets the current position (setxy [0 0])
    • arc -> creates an arc (eg: arc 180 100)
